TUI(Toast UI)를 사용하기 위해선 Bower 가 필요하고 Bower를 설치하기 위해선 npm(node package manager) 이 필요하다.

##npm 설치

{% highlight %}
http://nodejs.org
{% end highlight %}
![node install]({{ site.url }}/assets/node_install.PNG)

###npm 설치후 확인
{% highlight %}
$ node -v

$ npm -v
{% end highlight %}
![node install version]({{ site.url }}/assets/node_install_version.PNG)


##bower 설치

{% highlight %}
$ npm install -g bower
{% end highlight %}

> -g 옵션은 글로벌 설치 옵션이다

## TUI Chart(Toast UI 에서 TUI Chart로 예제로 해보겠다.)
[tui.chart](https://github.com/nhnent/tui.chart)
{% highlight %}
$ bower install tui-chart
{% end highlight %}

![tui install]({{ site.url }}/assets/tui-install.PNG)

{% highlight %}
<!DOCTYPE html>
<html lang="kr">

    <meta charset="UTF-8">
